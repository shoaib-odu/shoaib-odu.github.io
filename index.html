Retrieval Augmented Generation

The Retrieval Augmented Generation (RAG) application is a comprehensive tool designed to facilitate streamlined communication processes within the organization. It includes an admin dashboard and an agent view, allowing for efficient data management, chatbot creation, and FAQ generation. This application enhances productivity, communication, and customer support services through its intuitive interface and powerful functionalities.

Application Developer Intern - Resposibilities

Develop, Deploy, and Test

As an Application Development Intern, my role involved full-stack development using technologies such as React, Python, and NodeJS. Application Developer Intern - Expectations

A fully functional product

The expectations for my role were to contribute effectively to both frontend and backend development efforts for the RAG application. This included designing and implementing features such as data loading, chatbot creation, link sharing, data ingestion, chatbot testing, and FAQ generation.
